uv Cheat Sheet

Installation

curl -LsSf https://astral.sh/uv/install.sh | sh

Python Installation

uv python install 3.12

Virtual Environment

uv venv .venv
source .venv/bin/activate

Dependency Management

  • uv pip install <package> – Install a package
  • uv pip uninstall <package> – Uninstall a package
  • uv pip list – List installed packages
  • uv lock – Generate/update uv.lock
  • uv sync – Sync environment with pyproject.toml and uv.lock

Running Commands

uv run python manage.py runserver

Managing Dependency Groups

  • uv pip install <package> --group dev – Add to dev group
  • uv pip install <package> --group prod – Add to prod group
  • uv sync --group dev – Sync dev dependencies

Disposable Environments

uv tool run <tool>

Example:

uv tool run black .
